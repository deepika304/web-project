Project Name: Web Project – Consultation & Subscription Form

Overview:
A full-stack web app where users can submit a consultation form (Hero Form) and subscribe via a footer form.

Technologies:
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB (Mongoose)
- Deployment: Render (backend), Netlify or local browser (frontend)
- Middleware: CORS, Multer

Folder Structure:
- backend/
    ├─ app.js
    ├─ package.json
    ├─ routes/dataRoutes.js
    ├─ controllers/dataController.js
    ├─ models/FileModel.js
    ├─ models/SubscriptionModel.js
    └─ uploads/
- frontend/
    ├─ index.html
    ├─ admin.html
    ├─ style.css
    ├─ admin.css
    ├─ main.js
    └─ admin.js

Deployment Links:
- Backend: https://web-projectbackend.onrender.com
- Frontend: https://genuine-frangollo-847f04.netlify.app/

Instructions:
1. Open frontend URL in browser.
2. Fill the Hero Form or Footer Subscribe Form.
3. Submitted data is stored in MongoDB backend.
